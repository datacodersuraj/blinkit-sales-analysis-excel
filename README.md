# Blinkit Sales Analysis (Excel Project)

## 📊 Project Overview
This project analyzes Blinkit sales data using Microsoft Excel. The goal is to extract meaningful insights related to sales performance, product categories, and customer behavior.

## 🔧 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs

## 📁 Dataset
The dataset contains:
- Product details
- Sales records
- Category-wise performance

## 📌 Key Features
- Data cleaning and preprocessing
- Use of Pivot Tables for summary analysis
- Visualization using charts
- Sales trend analysis

## 📈 Insights
- Identified top-performing product categories
- Analyzed sales distribution across different segments
- Observed trends in customer purchasing behavior

## 🚀 How to Use
1. Download the Excel file
2. Open in Microsoft Excel
3. Explore pivot tables and charts for insights
